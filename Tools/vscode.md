# VS Code 常用快捷键速查表

## 光标与选择

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl + D` | 选中当前单词/下一个相同单词 | 连续按可选中多个相同内容，便于批量修改 |
| `Ctrl + Shift + L` | 选中所有相同单词 | 一次性选中文件中所有匹配项 |
| `Alt + 鼠标左键拖动` | 多光标选择 | 在多个位置同时输入 |
| `Ctrl + Alt + ↑/↓` | 向上/下添加光标 | 在相邻行创建多光标 |
| `Ctrl + U` | 撤销上一次选择 | 当 `Ctrl+D` 选多了时回退 |
| `Ctrl + Shift + →/←` | 按单词扩展选择 | 比按字符选择更快 |
| `Ctrl + L` | 选中当前行 | 快速选择整行内容 |
| `Ctrl + Shift + M` | 选中括号内内容 | 快速选中 `{}`、`()`、`[]` 内的代码块 |
| `Ctrl + Shift + \` | 跳转到匹配的括号 | 在括号对之间跳转 |

## 代码编辑

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Alt + ↑/↓` | 移动当前行/选区 | 上下移动代码行，无需复制粘贴 |
| `Shift + Alt + ↑/↓` | 复制当前行/选区到上/下方 | 快速复制代码 |
| `Ctrl + Shift + K` | 删除当前行 | 无需选中即可删除整行 |
| `Ctrl + /` | 注释/取消注释 | 行注释切换 |
| `Ctrl + Shift + /` | 块注释 | 添加 `/* */` 形式的注释 |
| `Ctrl + J` | 合并行 | 将下一行合并到当前行末尾 |
| `Ctrl + Enter` | 下方插入新行 | 无论光标在何处都在下方新建一行 |
| `Ctrl + Shift + Enter` | 上方插入新行 | 在当前行上方新建一行 |

## 导航与搜索

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl + P` | 快速打开文件 | 输入文件名跳转，无需鼠标找文件 |
| `Ctrl + Shift + F` | 全局搜索 | 在整个项目中搜索文本 |
| `Ctrl + G` | 跳转到指定行 | 输入行号快速定位 |
| `Ctrl + Shift + O` | 跳转到符号（函数/变量） | 在当前文件中快速导航 |
| `Ctrl + T` | 全局符号搜索 | 在整个项目中查找函数/类 |
| `Ctrl + B` | 显示/隐藏侧边栏 | 腾出更多编辑空间 |
| `` Ctrl + ` `` | 显示/隐藏终端 | 快速开关集成终端 |
| `F12` | 跳转到定义 | 查看函数/变量的定义位置 |
| `Alt + F12` | 速览定义 | 不跳转，小窗口预览定义 |
| `Ctrl + Alt + -` | 返回上一个位置 | 跳转后快速返回 |
| `Ctrl + Shift + -` | 前进到下一个位置 | 与上面配合浏览代码 |
| `Ctrl + Shift + Tab` | 浏览编辑器历史 | 在打开过的文件中后退 |

## 代码格式化与重构

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Shift + Alt + F` | 格式化文档 | 自动整理代码格式 |
| `Ctrl + K Ctrl + F` | 格式化选中区域 | 只格式化选中的代码 |
| `F2` | 重命名符号 | 安全重命名变量/函数，自动更新所有引用 |
| `Ctrl + .` | 快速修复 | 显示代码问题的修复建议 |
| `Ctrl + Shift + R` | 重构菜单 | 提取函数、变量等重构操作 |
| `Ctrl + Shift + .` | 聚焦到面包屑 | 快速导航文件结构 |

## 窗口与视图

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl + \` | 拆分编辑器 | 左右分屏对比代码 |
| `Ctrl + 1/2/3` | 切换到第1/2/3个编辑器组 | 在分屏间快速跳转 |
| `Ctrl + W` | 关闭当前标签页 | 快速关闭文件 |
| `Ctrl + Shift + T` | 重新打开关闭的标签页 | 误关文件时恢复 |
| `Ctrl + Tab` | 切换最近使用的文件 | 循环切换打开的文件 |
| `Ctrl + Shift + N` | 新建窗口 | 打开新的编辑器窗口 |
| `Ctrl + =/-` | 放大/缩小字体 | 调整编辑器字体大小 |
| `Ctrl + 0` | 重置字体大小 | 恢复默认字体大小 |

## 终端与调试

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `` Ctrl + ` `` | 显示/隐藏集成终端 | 快速开关集成终端 |
| `` Ctrl + Shift + ` `` | 新建终端 | 打开新的终端标签 |
| `Ctrl + PageUp/PageDown` | 切换终端标签 | 在多个终端间切换 |
| `F5` | 开始调试 | 启动调试会话 |
| `F9` | 切换断点 | 在当前行设置/取消断点 |
| `F10` | 单步跳过 | 逐行执行，不进入函数内部 |
| `F11` | 单步进入 | 进入函数内部执行 |
| `Shift + F11` | 单步跳出 | 执行完当前函数并跳出 |
| `Ctrl + Shift + F5` | 重新启动调试 | 快速重启调试会话 |
| `Shift + F5` | 停止调试 | 终止调试会话 |

## 智能提示与补全

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl + Space` | 触发建议 | 手动调出代码补全 |
| `Ctrl + Shift + Space` | 参数提示 | 查看函数参数信息 |
| `Tab` / `Enter` | 接受建议 | 确认使用当前补全项 |
| `Esc` | 关闭建议 | 取消代码补全窗口 |
| `Ctrl + I` | 触发内联提示 | Copilot 等 AI 补全触发 |

## Git 与版本控制

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl + Shift + G` | 打开源代码管理面板 | 快速查看 Git 状态 |
| `Ctrl + Enter` | 提交更改 | 在 Git 面板中提交 |
| `Ctrl + Shift + P` → `Git:` | Git 命令面板 | 执行各种 Git 操作 |

## 命令面板与设置

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl + Shift + P` / `F1` | 命令面板 | 执行所有可用命令 |
| `Ctrl + ,` | 打开设置 | 快速进入设置页面 |
| `Ctrl + K Ctrl + S` | 键盘快捷键 | 查看和自定义所有快捷键 |
| `Ctrl + K Ctrl + T` | 切换颜色主题 | 快速更换编辑器主题 |
| `Ctrl + K Z` | 禅模式 | 进入无干扰的全屏编辑模式 |

## 核心快捷键推荐（优先掌握）

以下 15 个快捷键能覆盖 80% 的高效编辑场景：

1. `Ctrl + D` - 批量选中相同内容
2. `Alt + ↑/↓` - 移动代码行
3. `Ctrl + P` - 快速打开文件
4. `Ctrl + Shift + F` - 全局搜索
5. `F12` - 跳转到定义
6. `F2` - 安全重命名
7. `Ctrl + /` - 快速注释
8. `` Ctrl + ` `` - 开关终端
9. `Ctrl + Shift + K` - 删除行
10. `Shift + Alt + F` - 格式化代码
11. `Ctrl + \` - 分屏编辑
12. `Ctrl + G` - 跳转到行
13. `Alt + F12` - 速览定义
14. `Ctrl + Shift + L` - 选中所有匹配
15. `Ctrl + Shift + P` - 命令面板

&gt; **提示**：在 VS Code 中按 `Ctrl + K Ctrl + S` 可打开快捷键设置，搜索任意命令查看或修改其快捷键。
