You are given a number of test cases, each with an integer \( n \). For each \( n \), compute the following sum for all \( n \)-permutations: 

1. For each permutation, count how many elements can be found by conjugating the permutation with all other permutations.
2. Raise this count to a fixed power \( k \).
3. Sum these values over all permutations.

Output the results modulo \( 998244353 \).
