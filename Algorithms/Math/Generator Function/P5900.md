无标号无根树计数问题
先考虑无标号有根树计数，无根树的计数可以从这个地方转化过来。

设$F(x)$是无标号有根树个数的OGF，$f_n = [x^n]F(x)$

我们可以枚举子树大小,而无标号树下得到递推关系：
$f_n = [x^n]x \cdot Euler(F(x))$
其中$Euler$是$Euler$变换，它表示生成的无标号多重集计数的OGF
有公式：
$$Euler(F(x)) = \prod_{n \geq 1} \left(\frac{1}{1 - x^n}\right)^{f_n}$$
$$Euler(F(x)) = \exp(\sum_{n \geq 1} \frac{F(x^n)}{n})$$
这两个公式是等价的，形式化的证明可以先 $\ln$ 再 $\exp$ 回去来证明。
第一个公式是比较直观的，实际上就是枚举每个大小的元素各自选了多少遍，组合起来是无序的。
第二个公式比较难以理解，需要理解EGF理论和$Polya$定理等等。
以后或许会针对这一块专门讲讲。

回到这一道题，我们可以得到
$$F(x) = x \cdot \exp(\sum_{n \geq 1} \frac{F(x^n)}{n})$$
两边取 $ln$
$$\ln F(x) = \ln x +  \sum_{n \geq 1} \frac{F(x^n)}{n}$$
$\ln$不能处理，直接求导
$$ \frac{F'(x)}{F(x)} = \frac{1}{x} + \sum_{n \geq 1} x^{n - 1} F'(x^n)$$
两边乘上$xF(x)$
$$F'(x) = F(x) + F(x)\sum_{n \geq 1}x^nF'(x^n)$$
把$\sum_{n \geq 1}x^nF'(x^n)$看作$G(x)$,枚举$n$的因子可以得到计算$g_n$的式子：
$$g_n = \sum_d [x^{n - d}] F'(x^d)= \sum_{d|n} \frac{n}{d}f_{\frac{n}{d}} =  \sum_{d|n} df_{d}$$
这个系数显然可以$O(n\log n)$计算完$g_1 \sim g_n$(也可以$O(n \log \log n)$计算)
于是有：
$$F'(x) = F(x) + F(x)G(x)$$
取$[x^n]$，
$$nf_n = f_n + \sum_{i = 1}^{n - 1}f_i g_{n - i}$$
$$f_n = \frac{1}{n - 1} \sum_{i = 1}^{n - 1}f_i g_{n - i}$$
这显然是一个半在线卷积形式了，可以直接分治ntt求解。

我们求完$f$后，再看怎么转化到无根树计数的问题上来。
实际上，我们只需要定下无根树的根就行了，结果就是对应的有根树的数量。
而重心直接和树的$size$相关，而且至多只有2个，非常合适。
根据重心的判据，我们指定一个点作为树的重心，那么不存在$size \geq n/2$的子树。于是我们可以枚举这样的$size \geq n / 2$ 的子树去容斥。
有：
$$h_n = f_n - \sum_{i = n / 2}^{n - 1}f_i  f_{n - i}$$
然而 $n$ 为偶数时会有问题，我们重复计算了。
修正一下：
$$h_n = f_n - \left(\sum_{i = n / 2 + 1}^{n - 1}f_i  f_{n - i}\right) - \binom{f_{n / 2}}{2}$$
