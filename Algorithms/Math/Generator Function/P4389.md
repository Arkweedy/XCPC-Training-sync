关键点：
$$\ln(1-x^k) \mod x^m$$
只有$\frac{m}{k}$个有效项。

直接使用级数展开即可：
$$\begin{align*}
\ln(1-x^k) = -\sum_{i = 1}^\infty \frac{x^{ik}}{i}
\end{align*}$$
于是
$$\begin{align*}
\ln\frac{1}{1-x^k} = \sum_{i = 1}^\infty \frac{x^{ik}}{i}
\end{align*}$$
而本质不同的背包物品只有$n$个，且至少是$1,2,\dots,n$
于是有效项的总数是$O(n\log n)$级别的。
最后做一遍多项式exp和多项式求逆即可。

或者直接引入无标号计数下的Euler变换：
$$Euler(F(x)) = \prod_{i = 1} \left(\frac{1}{1 - x^i}\right)^{f_i}$$
其中$f_i = [x^i]F(x)$
直接生成$F$的无标号计数生成函数。
