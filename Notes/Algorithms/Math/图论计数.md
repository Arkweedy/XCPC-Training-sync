# 图结构
## 有标号无向连通图 $C(x)$
考虑有标号无向图(不要求连通)的EGF：
$G(x) = \sum\limits_{n} 2^{\binom{n}{2}} \frac{x^n}{n!}$
而它是有标号无向连通图的生成集族，于是有关系：
$$G(x) = \exp(C(x)) \Rightarrow C(x) = \ln G(x)$$
## 有标号有向无环图 $D(x)$
这里图不一定联通
令$d_0 = 1$, 这是合理的，不能等于 $0$。
我们考虑枚举源点个数，有：
$$d_n = \sum_{i = 1}^{n} (-1)^{i + 1} \binom{n}{i}2^{i(n - i)}d_{n - i}$$
为什么这里有一个交错级数的容斥在这里？
因为对于一个实际上有$k$个源点的DAG，会被$d_{n - i}$ 贡献 $\binom{k}{i}$ 次，引入这个交错级数后，根据经典的二项式定理:
$$0 = (1 - 1)^k = \sum_{i = 0}^k (-1)^{i}\binom{n}{i} = 1 - \sum_{i = 1}^k (-1)^{i + 1}\binom{k}{i} \\
\Rightarrow \sum_{i = 1}^k (-1)^{i + 1}\binom{k}{i} = 1$$
于是我们确保对于$n$个点，恰有$k$个源点的DAG恰好被精确贡献一次。
ChatGPT也给了一种基于标准的交并容斥的理解方案：
https://chatgpt.com/share/6918552d-063c-8007-aa13-ce371ce9da77

这个自卷积式应该可以使用半在线卷积$O(n \log^2 n)$的复杂度解决。
进一步地，我们去优化这个式子，把他变成更优美的形式。
首先给出一个难以注意的恒等式：
$$ab = \binom{a + b}{2} - \binom{a}{2} - \binom{b}{2}$$
于是进行化简：
$$\begin{align*}    
    \frac{d_n}{n! 2^\binom{n}{2}} = \sum_{i = 1}^{n} \frac{(-1)^{i + 1}}{i! 2^\binom{i}{2}} \frac{d_{n - i}}{(n - i)! 2^\binom{n - i}{2}} \\
    \Rightarrow \sum_{i = 0}^n \frac{(-1)^{i + 1}}{i! 2^\binom{i}{2}} \frac{d_{n - i}}{(n - i)! 2^\binom{n - i}{2}} = [n = 0] \\
    def:\mathcal{L}(F)(x) = \sum_{n} \frac{f_n}{2^\binom{n}{2}}x^n \\
    \Rightarrow \mathcal{L}(e^{-x}) \mathcal{L}(D(x)) = 1
\end{align*}$$
其中$\mathcal{L}$是我们定义的一种线性变换：
$$\mathcal{L}(F)(x) = \sum_{n} \frac{f_n}{2^\binom{n}{2}}x^n $$
于是我们就可以得到$D(x)$:
$$D(x) = \mathcal{L^{-1}}\left(\frac{1}{\mathcal{L(e^{-x})}}\right)$$

## 有标号有向连通图 $S(x)$
在DAG计数的基础上，我们再来讨论有标号有向连通图计数问题。
根据blog https://codeforces.com/blog/entry/115617
我们可以把DAG中的每个点，进一步地变成具有某种特征的强连通分量，这与最后生成的图结构缩点后形成的DAG是一一对应的。
于是我们代入有标号有向连通图的EGF $S(x)$，对于的应该是所有有向连通图的EGF $A(x) = \sum_n 4^\binom{n}{2} \frac{x^n}{n!}$
于是我们得到新的表达式：
$$\mathcal{L}(A(x)) = \frac{1}{\mathcal{L}(e^{-S(x)})}$$
于是有：
$$S(x) = -\ln \mathcal{L^{-1}}\left(\frac{1}{\mathcal{L^{-1}}(e^x)}\right)$$
这里代入了$A(x) = \mathcal{L^{-2}}(e^x)$
