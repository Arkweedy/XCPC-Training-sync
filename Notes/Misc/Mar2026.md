## 1854C
和卡特兰数有些关系
其实只用考虑每相邻的两个数，在哪个位置合并即可，其他的数合并到前一个/后一个合并到其他数上是不影响它们的合并对答案的贡献的。
只考虑两个数的情况下，它们两个谁先后移是等概率的，为$\frac{1}{2}$
合并在$i$,那么可以减少 $n - i + 1$ 步操作
若两个数相距$d$，我们可以枚举后一个数还移动了 $k$ 步
设$f_d(k)$是相距为$d$的两个数，在距离后一个数$k$步处第一次相遇的概率，
这显然是一个类似于卡特兰数的东西，可以建立这样一个模型：
类比正则括号串，由于我们需要**第一次**相遇， 那么最外层是一层匹配的括号串
那么个数就是长为$2(k + d - 1)$, 且前$d - 1$个都是左括号的正则括号串个数。

先讲一下怎么$O(k^2)$求解
显然 $f_d(0) = \frac{1}{2^d}$
对于后移 $k$ 次的情况，我们容斥处理，可以通过它们第一次相遇在$j < k$处来分类非法情况， 而这个的次数我们先前是求解过的。相遇后，后面是随意排布的，直接组合数即可。
去掉提前相遇的情况，剩下一定是刚好第一次在$k$相遇的情况，因为我们限定了两种操作的种类。
于是有：
$$f_d(k) = \frac{1}{2^{d + 2k}} \left(\binom{d + 2k}{k} - \sum_{j = 0}^{k - 1} f_d(j)\binom{2(k - j)}{k - j}\right)$$
直接对每个$k$暴力计算是$O(n^2)$的，对本题来说可以通过 ，总时间复杂度为$O(n^2m)$

这个类似卡特兰数似乎还可以直接反射容斥$O(1)$计算？
考虑上面的模型，实际上就是从$(d - 1, d - 1)$到$(2(d + k - 1), 0)$，只能朝右上/右下走，且要求纵坐标大于等于$0$的格路计数
根据反射容斥，相当于没有纵坐标限制下， $(0, d - 1) \rightarrow (d + 2k - 1, 0)$ 的方案数减去 $(0, -d - 1) \rightarrow (d + 2k - 1, 0)$ 的方案数
列出式子：
$$f_d(k) = \binom{d + 2k - 1}{k} - \binom{d + 2k - 1}{k - 1}$$

好像也不是很难推,下次准备直接问AI之前先自己推推看好了，这种可以转化为坐标限制下的格路计数模型还是得想到反射容斥的好。

## AGC028 B
考虑计算每个位置的总贡献次数，可以转化为求解期望贡献次数，最后再乘上$n!$
有两种理解并求解一个位置的贡献次数的方法

### 考虑递推
考虑一个位置贡献多少次，其实就是对包含这个位置的段进行随机删除，多少次会被删掉
可以发现删除前缀/后缀是独立的(除开删除这个数自己)
对一个长为$i$的前缀段考虑删完一次还剩多长，有递推式：
$$f_0 = 0, f_1 = 1$$
$$f_i = \frac{1}{i}\sum_{j = 0}^{i - 1}f_{j} + 1$$
推导可以发现$f$就是调和数$H_n = \sum_{i = 1}^n \frac{1}{i}$ (不发现也能做)
