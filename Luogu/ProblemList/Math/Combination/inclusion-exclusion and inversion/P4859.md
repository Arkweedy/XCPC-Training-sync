和P6478非常像
实际上就是一个二分图$\{V_A,V_B\}$，权值有$a_i < b_j$的点有一条边，求$k-match$的数量
考虑到这个二分图的特殊性(边的存在有一定的传递性？)，我们可以想到将两边的点按照权值排序，这样处理之后，我们就可以知道$a_i$一定连向$a_{i - 1}$连向的所有点。
对于求恰好$k-match$，使用经典容斥—— 我们求出钦定$i$组match，其他任意的方案数
设恰好$i$组为$F(i)$,钦定$i$组为$G(i)$,有
$$G(k) = \sum_{i = k}^{n} \binom{i}{k} F(i)$$
反演：
$$F(k) = \sum_{i = k}^{n} (-1)^{i - k} \binom{i}{k} G(i)$$

对于怎么求$G$,我们使用前面的推论dp
设$dp_{i,j}$为使用$a_1 \sim a_i$,钦定$j$组match的方案数,$c_i$是比$a_i$小的$b$中元素的个数，有状态转移公式：
$$dp_{i,j} = dp_{i - 1,j} + \max(0,c_i - (j - 1))\times dp_{i - 1, j - 1}$$

直接滚动dp，取最后的dp数组，乘上自由match的阶乘因子作为$G$即可。
