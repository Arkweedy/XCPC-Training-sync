同样是容斥推式子，不过需要处理边界
经典$F(i,j)$是刚好$i$行，$j$列符合条件，$G(i,j)$是钦定$i$行，$j$列符合条件
那么有
$$G(x,y) = \sum_{i = x}^n\sum_{j = x}^n \binom{i}{x} \binom{j}{y} F(i,j)$$
反演一下
$$F(x,y) = \sum_{i = x}^n\sum_{j = x}^n (-1)^{n - i + n - j}\binom{i}{x} \binom{j}{y} G(i,j)$$

对于$G(i,j)$怎么求
设$c(i,j)$是对符合条件的$i$行，$j$列染色的方案数
很容易得到
\[
c(i,j) = 
\begin{cases}    
    3^{j + n(n - j)} &i = 0\\
    3^{i + n(n - i)} &j = 0\\
    3^{1 + (n - i)(n - j)} &others
\end{cases}    
\]
有$$G(i,j) = \binom{n}{i} \binom{n}{j} c(i,j)$$

求每一种行列组合下的符合条件数显然不太好，考虑求出$0$行$0$列符合条件的作为非法状态，再用总方案减非法方案。
总方案$3^{n^2}$个
非法方案即
$$F(0,0) = \sum_{i = 0}^n\sum_{j = 0}^n (-1)^{i + j} G(i,j) = \sum_{i = 0}^n\sum_{j = 0}^n (-1)^{i + j} \binom{n}{i} \binom{n}{j} c(i,j)$$

直接计算是$O(n^2 \log n)$的，考虑对内层使用二项式定理
由于$c(i,j)$的取值在特殊点的计算方法不一样，我们考虑分开计算
对于$i \in [1,n]$
内层为
\[
\begin{align*}    
    &\sum_{j = 0}^n (-1)^j\binom{n}{j} c(i,j) \\
    & = \left(\sum_{j = 0}^n (-1)^j\binom{n}{j} 3^{1 + (n - i)(n - j)}\right) - 3^{1 + n(n - i)} + c(i,0)\\
    &= 3*(3^{n - i} - 1)^n - 3^{1 + n(n - i)} +3^{i + n(n - i)}
\end{align*}    
\]
而对于$i = 0$
直接计算得
$$\sum_{j = 0}^{n} (-1)^{j} \binom{n}{j} c(0,j) \\
= \sum_{j = 0}^{n} (-1)^{j} \binom{n}{j} 3^{j + n(n - j)}\\ = \sum_{j = 0}^{n} (-3)^{j} \binom{n}{j} (3^n)^{n - j}\\ = (3^n - 3)^n$$

这样计算就是$O(n \log n)$的，直接计算即可。
