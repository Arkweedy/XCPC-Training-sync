定义:
$f(i,j,k)$为恰好$i$行，$j$行，$k$个颜色没选
$g(i,j,k)$为至少$i$行，$j$行，$k$个颜色没选
$g$ 这里的“至少”的含义是：先钦定$i,j,k$三个限制对应的行，列，颜色，剩下的再任意，也就是说**允许对同一个方案重复计数**,只要钦定的方案不一样就计入一次
有
$$g(i,j,k) = \binom{n}{i}\binom{m}{j}\binom{c}{k}(c -k + 1)^{(n - i)(m - j)}$$

又根据定义，
$g(x,y,z) = \sum\limits_{i = x}^{n}\sum\limits_{j = y}^{m}\sum\limits_{k = z}^{c} \binom{i}{x}\binom{j}{y}\binom{k}{z}f(i,j,k)$
高维二项式反演:
$f(x,y,z) = \sum\limits_{i = x}^{n}\sum\limits_{j = y}^{m}\sum\limits_{k = z}^{c} (-1)^{i + j + k - x - y - z}\binom{i}{x}\binom{j}{y}\binom{k}{z}g(i,j,k)$

于是有答案：
\[
\begin{align*}
    ans 
    &= f(0,0,0)\\
    &= \sum\limits_{i = 0}^{n}\sum\limits_{j = 0}^{m}\sum\limits_{k = 0}^{c} (-1)^{i + j + k}\binom{n}{i}\binom{m}{j}\binom{c}{k}(c -k + 1)^{(n - i)(m - j)}
\end{align*}
\]

可以对答案式做进一步化简：
交换求和和调整指标枚举：
$$\sum\limits_{k = 0}^{c} (-1)^{k}\binom{c}{k}\sum\limits_{i = 0}^{n}(-1)^{n - i}\binom{n}{i}\sum\limits_{j = 0}^{m} (-1)^{m - j}\binom{m}{j} (c -k + 1)^{ij}$$
对最后一部分使用二项式定理：
$$\sum\limits_{k = 0}^{c} (-1)^{k}\binom{c}{k}\sum\limits_{i = 0}^{n}(-1)^{n - i}\binom{n}{i} ((c - k + 1)^i - 1)^m$$

这样化简后，相当于少了一层求和,$(c - k + 1)^i$在循环里顺手维护即可，最终时间复杂度 $\mathcal{O(cn \log(m))}$
当然也可以调整为 $\mathcal{O(cm \log(n))}$,取$n,m$大的一维使用二项式定理即可，这对$n,m,nm$范围同阶的题处理效果很好。

换一条路，使用斯特林反演
\[
H(T) = \sum_{k=0}^{c} (-1)^k \binom{c}{k} (c-k+1)^T
\]

\[
X^T = \sum_{r=0}^{T} {T \brace r} x^{\underline{r}}
\]
可得
\[
H(T) = \sum_{k=0}^{c} (-1)^k \binom{c}{k} \sum_{r=0}^{T} {T \brace r} (c-k+1)^{\underline{r}}
\]

\[
= \sum_{r=0}^{T} {T \brace r} \sum_{k=0}^{c} (-1)^k \binom{c}{k} (c-k+1)^{\underline{r}}
\]
定义
\[
I_r = \sum_{k=0}^{c} (-1)^k \binom{c}{k} (c-k+1)^{\underline{r}}
\]
换元
\[
I_r = \sum_{k=0}^{c} (-1)^{c - k} \binom{c}{k} (k+1)^{\underline{r}}
\]
定义前向差分：
\[
    \Delta f(x) = f(x + 1) - f(x)
\]
利用前向差分公式：
\[
    \Delta^cf(x) = \sum_{t = 0}^{c} (-1)^{c - t} \binom{c}{t}f(x + t)
\]
对$I_r$进行变换：
\[
    I_r = \Delta^c x^{\underline{r}} |_{x = 1}
\]
又对于下降幂的向前差分，有：
\[
    \Delta x^{\underline{r}} = rx^{\underline{r - 1}}\\
    \Delta^c x^{\underline{r}} = r^{\underline{c}}x^{\underline{r - c}}
\]
代入$I_r$，
\[
    I_r =  r^{\underline{c}}1^{\underline{r - c}} 
\]
$I_r$只有$r = c,c + 1$ 时不为 $0$
于是有：
\[
\begin{align*}
H(T) &= \sum_{r=0}^{T} {T \brace r} I_r\\
&=  {T \brace c} c! + {T \brace c + 1} (c+1)!
\end{align*}
\]


另外一个结论：
\[
\sum_{k=0}^{c} (-1)^{c-k} \binom{c}{k} k^T
\]

\[
= \sum_{k=0}^{c} (-1)^{c-k} \binom{c}{k} \sum_{r=0}^{T} {T \brace r} k^{\underline{r}}
\]

\[
= \sum_{r=0}^{T} {T \brace r}  \sum_{k=0}^{c} (-1)^{c-k} \binom{c}{k} k^{\underline{r}}
\]

\[
\sum_{k=0}^{c} (-1)^{c-k} \binom{c}{k} k^{\underline{r}} = \Delta^c k^{\underline{r}} |_{k = 0}
\]

\[
= r^{\underline{c}} \left. k^{\underline{r-c}} \right|_{k=0}
\]

当 \(r-c = 0\) 时，才不为 0，为 \(r^{\underline{r}} = r!\)

\[
\Rightarrow \text{原式} = {T \brace c} c!
\]

于是我们得到一个第二类斯特林数的计算公式(当然可能是循环证明)：
\[
\begin{align*}
{n \brace m} &= \frac{1}{m!} \sum_{j=0}^{m} (-1)^{m-j} \binom{m}{j} j^n \\
&=  \sum_{j=0}^{m}  \frac{(-1)^{m-j}j^n}{j!(m - j)!} 
\end{align*}
\]
