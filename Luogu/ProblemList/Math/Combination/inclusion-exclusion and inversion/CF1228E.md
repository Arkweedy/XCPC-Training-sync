和P6076的化简相似
将内层的求和用二项式定理写成闭式即可

定义：
$f(i,j)$为恰好$i$行，$j$列没有$1$的情况数
$g(i,j)$为钦定任意$i$行，$j$列没有$1$，其他随意的情况数(可重复计数)

有
$$g(i,j) = \binom{n}{i}\binom{n}{j} k^{(n - i)(m - j)}(k - 1)^{nm - (n - i)(m - j)}$$
根据定义
$$g(x,y) = \sum_{i = x}^{n} \sum_{j = y}^{n} \binom{i}{x}\binom{j}{y} f(i,j)$$

反演得
$$f(x,y) = \sum_{i = x}^{n} \sum_{j = y}^{n} (-1)^{2n - i - j}\binom{i}{x}\binom{j}{y} g(i,j)$$

答案为$f(0,0)$

$$f(0,0) = \sum_{i = x}^{n} \sum_{j = y}^{n} (-1)^{2n - i - j} \binom{n}{i}\binom{n}{j} k^{(n - i)(m - j)}(k - 1)^{nm - (n - i)(m - j)}$$

可$O(n^2)$计算
进一步化简：
枚举$n - i, m - j$,使用二项式定理
$$f(0,0) = \sum_{i = 0}^{n} \sum_{j = 0}^{n} (-1)^{i + j} \binom{n}{i}\binom{n}{j} k^{ij}(k - 1)^{nm - ij} \\
= (k - 1)^{nm}\sum_{i = 0}^{n} (-1)^{i} \binom{n}{i} \sum_{j = 0}^{n} (-1)^{j} \binom{n}{j} \left(\frac{k}{k - 1}\right)^{ij}\\
=(k - 1)^{nm}\sum_{i = 0}^{n} (-1)^{i} \binom{n}{i} \sum_{j = 0}^{n} \binom{n}{j} \left(\left(-\frac{k}{k - 1}\right)^{i}\right)^j 1^{n - j}\\
=(k - 1)^{nm}\sum_{i = 0}^{n} (-1)^{i} \binom{n}{i} \left(1 - \left(\frac{k}{k - 1}\right)^{i}\right)^n$$

这个式子计算是 $O(n\log n)$的
